## Projects

### Titanic Survival Prediction
https://github.com/Zishaniitm/Codsoft-Internship-Projects
> End-to-end machine learning project — from raw data to trained model

Built a binary classification model on the classic Titanic dataset. Covered the complete ML pipeline independently.

- Handled missing data with group-level median imputation (Pclass + Sex)
- Engineered 6 new features: `Title`, `FamilySize`, `IsAlone`, `AgeGroup`, `HasCabin`, `FareBand`
- Trained and compared 4 algorithms: Logistic Regression, Decision Tree, Random Forest, KNN
- Tuned hyperparameters with `GridSearchCV` (optimised for F1, not just accuracy)
- Evaluated with confusion matrix, precision, recall, F1, and ROC/AUC

**Result:** Random Forest — 84% accuracy · AUC 0.87  
**Stack:** Python · Pandas · Scikit-learn · Matplotlib · Seaborn
